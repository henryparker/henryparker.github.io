Inverse Transform Sampling

Inverse Transform Sampling is the simplest method for sampling from a parametric univariate distribution, based on inverse probability transform.

Feb 11, 2025
3 minute read

Background

Inverse Transform Sampling is a fundamental technique in probability theory and statistics, allowing us to generate random numbers from any desired probability distribution, given its cumulative distribution function (CDF) and its inverse. This method is particularly useful when we have access to a uniform random number generator but need samples from a more complex distribution.

Core Idea

Given a pseudorandom number generator that produces numbers $u\sim U(0,1)$, our goal is to sample from a target distribution $D$. The key assumption here is that we can easily compute both the cdf $F$ of $D$ and its inverse $F^{-1}$. Since the cdf $F$ is a monotonically non-decreasing function and maps value from the domain of $D$ to the interval $[0,1]$, we can use the inverse cdf $F^{-1}$ to map a uniform random number $u$ back to a value in the domain of $D$. This process is visually illustrated below.

Step-by-Step Process

  1. Generate a Uniform Random Number: Start by generating a random number $u\sim U(0,1)$.
  2. Compute the Inverse CDF: Determine the cdf of $F$ of the target distribution $D$ and compute its inverse $F^{-1}$.
  3. Map to the Target Distribution: Use the inverse cdf to map the uniform random number $u$ to a sample from $D$ by calculating $F^{-1}(u)$.
  4. Repeat: Repeat steps 1-3 until you have generated enough samples.

Note: For continuous distributions, each unique $u$ will map to a unique value in the domain of $D$. However, for discrete distributions, different $u$ values may map to the same value in the domain of $D$.

Example: Exponential Distribution

Let’s consider the exponential distribution, which is commonly used to model the time between events in a Poisson process.

The probability density function (PDF) of the exponential distribution is:

$$Expon(x|\lambda)\triangleq \lambda e^{-\lambda x}$$

The corresponding CDF is:

$$F(x)=1-e^{-\lambda x}$$

The inverse CDF (quantile function) is:

$$F^{-1}(p)=-\frac{\ln{(1-p)}}{\lambda}$$

To generate samples from the exponential distribution, we can follow these steps:

  1. Generate a uniform random number $u\sim U(0,1)$.
  2. Use the inverse CDF to map $u$ to a sample from the exponential distribution: $F^{-1}(u)\sim \text{Expon}(\lambda)$.

In fact, since $1-u\sim U(0,1)$, we can simplify the calculation to $-\ln{(u)}/\lambda$.

Math Proof

Theorem. If $u\sim U(0,1)$ is a uniform r.v., then $F^{-1}(U)\sim F$.

Proof.

\[ \begin{align*} Pr(F^{-1}(U)\leq x)&=Pr(U\leq F(x))\\ &=F(x)\ (\text{because } Pr(U\leq y)=y) \end{align*} \]

Limitations of Inverse Transform Sampling

While Inverse Transform Sampling is a powerful method, it has some limitations:

  • Complexity of the Inverse CDF: The method relies on the ability to compute the inverse of the CDF. For many distributions, such as the normal or Weibull distributions, the CDF does not have a simple closed-form expression, making it difficult to compute the inverse. In such cases, numerical methods or approximations are required, which can increase computational complexity.
  • Scalability in High Dimensions: Inverse Transform Sampling is not well-suited for high-dimensional problems. As the dimensionality increases, handling the marginal distributions and their dependencies becomes increasingly complex, making this method less practical.
  • Independence Assumption: This method assumes that the random variables are independent. For correlated variables, other techniques like Markov Chain Monte Carlo (MCMC) or Accept-Reject Sampling are more appropriate.
